Web Development
===============
1. FrontEnd : User Interface ( Look & Feel )
        HTML : Content ( div, paragraph, table, list, form , audio)
        CSS :  Styling ( border , background , width, height, margin, padding)
        Bootstrap : CSS Library , Pre-defined css classes
        Javascript : Behaviour of a webpage
        Angular :  Javascript Framework ( RAD, CLI , Form, Routing, AJAX)

2. BackEnd : Develop APIs ( Java , .net, Python , PHP , Nodejs)

3. DataBase : Holds the data ( oracle , mysql , mongodb )



First HTML Program
===================
1. create an HTML file ( program_1.html )

2.  !  ( Emmet / Shortcuts ) add boilerplate code
    https://docs.emmet.io/cheat-sheet/

3. Add Actual HTML, CSS , Javascript code in that file

4. To Open/Run HTML file in browser 
    a. install vscode extension ( open in browser / live server )
    b. alt+b  / alt+shift+b
