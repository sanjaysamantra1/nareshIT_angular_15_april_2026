Variable Scopes
===============
1. Global : variables declared outside functions, they can be used by any of the functions.
2. Function : variables declared inside a function, can be accessed by that function only
3. Block : variables declared inside a block, can be accessed by that block only



Strict Mode
============
-in strict mode variable declaration without var/let/const is not allowed.
-in strict mode, function with duplicate param is not allowed.



Hoisting
========
-behavior of moving declarations to the top of the scope during compilation.
-creating memory location for variables during compilation phase.
-During Compilation phase for all(var/let/const) variables memory is created.



Temporal Dead Zone
==================
-even though let & const variables are hoisted( memory gets created during compilation)
 we cannot access those variables as slong as actual values are not initialized,
